What it is

Bunkforces is a local-first web tool that pulls random Codeforces problems matching your filters, typesets them into a compact, print-ready PDF you can carry to class, and keeps per-problem notes and statuses in your browser so you can implement the solutions at home.

Features

  • Problem picker — pick any number of random problems, filtered by rating range, division (Div 1–4 / Educational), problem index range, tags (match any / all), minimum solve count, and solved / unsolved (via your CF handle). Reroll individual problems; duplicates are never picked twice.
  • PDF typesetter — a clean, single-column, ink-friendly print-to-PDF with adjustable font size, margins, header-only vs. full statement, sample tests, and ruled / dotted / blank scratch space under each problem. Every problem carries its contest, index, rating, tags, URL and a QR code.
  • Notes & local storage — per-problem notes and status (To-do / Thinking / Solved-on-paper / Implemented), named saved problem sets, and JSON export / import. Nothing leaves your browser.

Data & privacy

Problem data comes from the Codeforces public API and is cached in your browser for offline use. Your notes, statuses and saved sets live only in this browser's local storage — there is no account and no server storing your data. Use Export to back them up as JSON.
